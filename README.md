# Used_Car_Price_Analysis_R
This project shows , clean messy data  , analyze business questions ,  visualize insights , build prediction models using  car sales data
Used Car Price Analysis in R

Tools Used
- R
- RStudio
- ggplot2
- dplyr

Project Goals
Analyze used car sales data to identify price trends and build a prediction model.

Key Insights
- Toyota and Ford have highest sales
- Mileage strongly affects price
- Newer cars have higher resale value
